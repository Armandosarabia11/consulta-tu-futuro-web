# Consulta tu futuro INC – Landing Page

Este proyecto consiste en el desarrollo de una **landing page monolítica** para la consultoría financiera **Consulta tu futuro INC**, creada como parte de una práctica académica de diseño web utilizando inteligencia artificial como herramienta de apoyo.

## Descripción del proyecto

La página presenta una consultoría financiera enfocada en educación financiera, planificación del ahorro, inversiones, seguros e hipotecas. El objetivo principal de la web es **captar leads interesados en servicios de asesoría financiera personalizada**.

El sitio ha sido diseñado como una **landing page de una sola página**, donde el usuario puede navegar por las diferentes secciones mediante desplazamiento interno (anclas).

## Tecnologías utilizadas

El proyecto se desarrolló utilizando tecnologías web básicas:

- **HTML5 semántico**
- **CSS**
- Diseño **mobile-first**
- Metadatos SEO
- Open Graph para redes sociales
- Datos estructurados con JSON-LD

No se utilizaron frameworks externos ni JavaScript, cumpliendo con los requisitos de la práctica.
